🚀 HOW TO USE

1. CUSTOMIZING AI BADGE TEXT:
   Edit index.html, find: <span class="ai-badge">Exploring Gen AI</span>
   Change to: <span class="ai-badge">Your Text Here</span>

2. ADDING NEW NEWS ITEMS:
   Edit index.html, find #ai-news section
   Copy/paste a news-item div and update:
   - news-date
   - news-title
   - news-source

3. CONNECTING TO REAL API:
   Create js/ai-news-api.js to fetch from NewsAPI or RSS feeds
   Update js/main.js to call the new module

4. CHANGING COLORS:
   Edit css/base.css :root section:
   --rust: #c0392b;    (primary accent)
   --gold: #c9a84c;    (secondary accent)
