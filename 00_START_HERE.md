## 🔧 Customization

### Change AI Badge Text
Edit `index.html`:
```html
<span class="ai-badge">Your Custom Text</span>
```

### Update News Items
Edit the AI news section in `index.html`:
```html
<div class="news-item">
  <div class="news-date">Your Date</div>
  <div class="news-title">Your Title</div>
  <div class="news-source">Your Source</div>
</div>
```

### Connect Real News API
Create a new JS module to fetch from NewsAPI.org or similar
